# 查看论文索引

查看所有已保存的论文，支持按方向筛选。

**用法：**
- `/list-papers` — 查看所有方向的论文统计
- `/list-papers <方向>` — 查看指定方向的所有论文
- `/list-papers <方向> track` — 只看追踪到的新论文
- `/list-papers <方向> survey` — 只看调研的重要论文

**示例：**
- `/list-papers`
- `/list-papers 3D模型生成`
- `/list-papers 神经辐射场 survey`

---

## 执行步骤

你是一个帮助计算机图形学博士管理论文索引的助手。所有输出请使用**中文**。

参数：**$ARGUMENTS**

### 第一步：读取索引

读取 `D:/claude_tools/paper_search/papers_db.json`。

同时读取 `D:/claude_tools/paper_search/paper_tracker_state.json` 以获取各方向的最后查询时间。

若 papers_db.json 不存在或为空，提示用户先使用 `/track-papers` 或 `/survey-papers` 添加论文。

### 第二步：解析参数

解析 `$ARGUMENTS`：
- 若参数为空：显示所有方向的统计概览
- 若参数为 `<方向>`：显示该方向的所有论文
- 若参数为 `<方向> track`：只显示 type="track" 的论文
- 若参数为 `<方向> survey`：只显示 type="survey" 的论文

方向名称支持模糊匹配。

### 第三步：展示结果

**情况A：无参数（概览模式）**

```
## 论文索引概览

| 研究方向 | 论文数 | 最后查询 |
|---------|--------|---------|
| 3D模型生成 | 15篇 | 2026-03-05 |
| 神经辐射场 | 23篇 | 2026-02-20 |
| ...      | ...   | ...      |

**共 <N> 个方向，<M> 篇论文**

提示：使用 `/list-papers <方向>` 查看某方向的详细论文列表。
```

**情况B：指定方向**

```
## <方向> 论文列表
**共 <N> 篇**（追踪: X篇 | 调研: Y篇）

--- 追踪论文（按时间倒序）---

1. **<标题>**
   - 作者：<作者>
   - 日期：<YYYY-MM> | 发表于：<venue>
   - 摘要：<一句话摘要>
   - 收录于：<found_at>

--- 调研论文（按影响力排序）---

...
```

按 type 分组展示，track 类按 found_at 倒序，survey 类按 date 倒序。
